功能特性
短线精灵，永远在场

不需要盯着大盘，StockRadar 帮你守着全市场 5000+ 只股票，有异动第一时间推到眼前。

🚀
实时异动推送

WebSocket 连接本地行情引擎，毫秒级推送火箭发射、高台跳水等异动信号。

📌
永远置顶浮窗

无边框半透明设计，永远置顶，不遮挡其他窗口，随时可见不打扰工作。

⚙️
自定义方案

自由配置筛选方案，设定市值范围、涨幅阈值、成交量条件，精准过滤噪音。

📊
市场情绪面板

实时显示沪指行情、涨跌家数、板块热度排名，一眼看清市场情绪。

🔍
复盘扫描

盘后复盘模式，用自定义方案扫描全市场历史快照，找出符合条件的个股。

💾
本地数据缓存

日线数据本地 SQLite 存储，离线可查历史，不依赖外部服务，数据完全自主。

信号类型
四种核心异动信号

覆盖短线最常见的四类异动形态，每种信号都有独立的参数配置。

🚀
火箭发射

短时间内快速拉升，涨幅超过设定阈值。捕捉主力资金快速建仓或游资爆炒的瞬间。

🏊
高台跳水

高位快速下跌，跌幅超过设定阈值。及时预警主力出货或利空突袭，规避风险。

📊
放量突破

成交量异常放大，配合价格突破关键位置。量价配合是主力入场的重要信号。

🔒
涨跌停

实时监控涨停、跌停状态，包括封板、炸板等关键节点，第一时间掌握极端行情。

本地优先，数据自主

Electron 桌面壳 + Python 行情引擎，所有数据本地处理，不经过任何第三方服务器。
